# example-pulumi

Some basic examples for Pulumi using different providers.

## Azure
* [Getting Started](https://www.pulumi.com/docs/get-started/azure/)

Export the following values
```
export ARM_CLIENT_ID="..."
export ARM_CLIENT_SECRET="..."
export ARM_SUBSCRIPTION_ID="..."
export ARM_TENANT_ID="..."
```

## AWS
* [Getting Started](https://www.pulumi.com/docs/get-started/aws/)

Export the following values
```
export AWS_ACCESS_KEY_ID="..."
export AWS_SECRET_ACCESS_KEY="..."
export AWS_REGION="..."
```
